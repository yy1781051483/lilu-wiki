## 核心摘要

芒格是李录投资生涯和人生哲学的最重要影响者。李录最受震动的不是芒格的投资技术，而是他的人生境界——在妻子去世、几乎完全失明等接连打击后，始终不见悲观绝望，宠辱不惊，"不以物喜，不以己悲"。李录认为芒格身上完整体现了西方启蒙运动以来"理性、科学、人文、进步"的基本精神。芒格对巴菲特最大的贡献是推动其从格雷厄姆的"烟蒂股"转向寻找具有经济护城河的优质公司——常劲将此概括为"从猿变成了人"。

## 思想体系核心（在本Wiki中的贡献）

1. **护城河理论的实际推动者** — 巴菲特的护城河框架实际由芒格推动形成，使价值投资从资产折价升级为优质公司的长期持有。

2. **跨学科多元思维模型** — 在脑中构建来自历史、心理、生物、数学、工程、物理、经济等学科的思维模型；Lollapalooza效应（多种思维模型叠加共振）。

3. **知识的诚实（Intellectual Honesty）** — 常劲、李录均将此归为芒格与李录共同强调的核心品性，知道自己知道什么、不知道什么。

4. **极度的耐心+极度的果断** — 李录在2019年北大演讲中引用：芒格看《巴伦周刊》40年只发现1个机会，"看了三十多年之后才发现这么一个机会，发现这个机会之后又过了十年，还没发现第二个。但是这并不影响他"。

5. **人生境界：宠辱不惊** — 李录在2018年补充采访中描述：芒格在接连打击（失明、丧偶）后始终未见悲观，体现了一种彻底的宁静与从容。

6. **"在有鱼的地方钓鱼"** — 芒格以钓鱼为喻的核心投资原则，常劲在北大课程中频繁引用。

## 时间线

| 年份 | 事件 |
|---|---|
| 1924 | 出生，美国内布拉斯加州奥马哈 |
| 约1978年 | 成为伯克希尔哈撒韦副董事长，与巴菲特建立长期合伙关系 |
| 约2000年代初 | 推动巴菲特从"烟蒂股"转向"护城河股"（李录描述） |
| 2003年 | 芒格家族开始通过喜马拉雅资本持有比亚迪，与李录建立合伙关系 |
| 2008年 | 支持伯克希尔以8港元/股入股比亚迪（帮助与Costco建立背书，促成口罩首笔6000万只订单） |
| 2018年 | 在奥马哈接受《红周刊》联合采访，称与李录的合作是人生三笔投资之一（时年95岁） |
| 2021年 | 99岁时买入人生最后一只股票，并亲眼看到翻倍 |
| 2023年11月 | 感恩节周末于家中平静离世，周四晚与家人吃火鸡，周日离世（李录2026年演讲中描述） |
