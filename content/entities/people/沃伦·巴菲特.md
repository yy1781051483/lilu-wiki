# 沃伦·巴菲特

## 基本信息

| 字段 | 内容 |
|---|---|
| 全名 | Warren Edward Buffett |
| 中文名 | 沃伦·巴菲特 |
| 生年 | 1930年8月30日 |
| 身份 | 伯克希尔哈撒韦董事长兼CEO；本杰明·格雷厄姆的学生；价值投资体系的集大成者 |
| 与李录关系 | 引领李录进入价值投资的直接启蒙者；2008年通过伯克希尔入股比亚迪，成为中美资本市场的重要纽带 |
| 与常劲关系 | 思想传承源头；常劲在奥马哈股东大会持续观察，2024年和2025年均发表评述 |

## 核心摘要

巴菲特对李录的影响始于1990年代初的一次哥伦比亚大学演讲——李录误以为是"免费午餐"而前往，由此进入价值投资领域，从未动摇。巴菲特在李录体系中代表的是"优秀公司长期持有"的进化——相对于格雷厄姆的烟蒂股，巴菲特（在芒格推动下）转向寻找具有持续竞争优势的公司。李录将伯克希尔成功的最难复制之处定义为巴菲特对个人利益的彻底淡泊：50年工资不超过10万美元/年，无奖金，无期权，从未增发新股。2025年股东大会，巴菲特宣布卸任，常劲认为这是"价值投资大道的新起点"。

## 思想体系核心（在本Wiki中的贡献）

1. **价值投资的集大成与普及** — 格雷厄姆建立基础理论，巴菲特（在芒格影响下）将其从"资产折价"进化到"优秀企业长期持有"，并通过数十年伯克希尔年报向全球传播。

2. **伯克希尔的结构性示范** — 李录以伯克希尔的费率结构（无管理费、无期权）为喜马拉雅资本的模板；常劲多次在课程和演讲中以1983年巴菲特致股东信作为学生作业材料。

3. **市场先生/投票机与称重机** — 巴菲特将格雷厄姆的"市场先生"比喻进一步发展为"市场短期是投票机，长期是称重机"，被常劲列为市场先生七条结论之第六条。

4. **"在有鱼的地方钓鱼"** — 巴菲特/芒格共同体现了专注少数最优机会的集中投资哲学，是喜马拉雅资本投资方法论的核心参照。

5. **比亚迪的背书** — 2008年伯克希尔入股比亚迪，是李录推动的，也是巴菲特/伯克希尔在中国企业上最具代表性的公开背书，被李录定义为"中美资本市场互信的重要纽带"。

## 时间线

| 年份 | 事件（与李录/常劲相关） |
|---|---|
| 约1990年代初 | 在哥伦比亚大学演讲，李录误入，由此进入价值投资 |
| 2006年 | 李录在哥伦比亚商学院（格雷厄姆价值投资中心）演讲，延续巴菲特传承 |
| 2008年 | 伯克希尔以8港元/股入股比亚迪（李录推动）；金融危机中向高盛等投资（常劲引用案例） |
| 2018年 | 在奥马哈接受《红周刊》联合采访（与芒格、李录同场） |
| 2024年 | 芒格去世后，巴菲特在股东大会首次独挑主场，C位让给格雷格·阿贝尔（常劲观察） |
| 2025年 | 宣布卸任伯克希尔CEO，常劲称之为"价值投资大道的新起点" |
