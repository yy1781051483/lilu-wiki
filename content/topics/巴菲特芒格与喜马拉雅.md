# 巴菲特芒格与喜马拉雅

## 摘要

查理·芒格和沃伦·巴菲特对李录和常劲的影响，是理解喜马拉雅资本思想体系的关键背景。芒格通过15年（2003年起）的合伙关系对李录产生了深刻的人生与投资双重影响；巴菲特通过1990年代初的哥伦比亚大学演讲引领李录进入价值投资领域，并于2008年通过伯克希尔入股比亚迪成为双方的历史性纽带。常劲则通过参与芒格书院、翻译《穷查理宝典》等方式，在喜马拉雅资本内部承担着传承和传播这一思想体系的角色。

## 李录/常劲的核心立场

**李录**：芒格的影响超越投资技术层面，是人生境界的示范——"不以物喜，不以己悲"，完整体现了西方启蒙运动以来"理性、科学、人文、进步"的基本精神。巴菲特是这一精神的另一个化身，其对个人利益的彻底淡泊（50年工资不超过10万美元/年）是伯克希尔成功最难被复制的核心。

**常劲**：芒格是"理性的化身"，其"极致诚实"（不管是在人前还是人后，做的事情从来都是一样的）是最打动他的品质。价值投资的历史演变（从格雷厄姆的"烟蒂股"到巴菲特/芒格的"护城河股"）是一次"从猿变成人"的跨越。

## 关键论点与证据

### 李录与巴菲特的初遇（1990年代初）

李录约1990年代初在哥伦比亚大学误以为巴菲特演讲是"免费午餐"而前往，由此进入价值投资领域，从此未曾动摇。他后来在多个场合描述这一"偶然性"：

> "价值投资很多时候就像打疫苗，要么管用要么不管用，打下去马上就知道。"
> — [[sources/li/【文章目录】/1-李录核心文字稿/06-2020-李录-诺亚访谈|06-2020-李录-诺亚访谈]]（2020年，李录）

### 芒格对李录的人生影响（李录，2018年）

李录描述了他与芒格长达15年的合伙关系（2003年起）。他最受震动的不是投资技术，而是芒格的人生境界：

> "他讲的是什么都是真相。真相就是公司本身的真实价值，天然的价值。"
> — [[sources/li/【文章目录】/1-李录核心文字稿/03-2018-李录-红周刊采访|03-2018-李录-红周刊采访]]（2018年，李录）

李录对芒格晚年遭遇（妻子去世、几乎完全失明）的描述：

> "他们希望复制成功来赚取自己的利益，却不知道正是因为这种对个人利益的淡泊，才成就了长期卓越的业绩。只有这样，才能比较从容、自然地去布置投资节奏，不受制于任何人，没有任何压力。"
> — [[sources/li/【文章目录】/1-李录核心文字稿/04-2018-李录-红周刊补充采访|04-2018-李录-红周刊补充采访]]（2018年，李录）

### 伯克希尔成功的深层原因（李录分析）

李录深入分析了伯克希尔成功的结构性原因：巴菲特50年工资从未超过每人10万美元，无奖金，无期权，所有股票均在市场上用自己的钱买入，从未增发新股。李录以此为模板建立喜马拉雅资本的费率结构（不收管理费，前6%收益免费）。

芒格的三笔投资（芒格本人在2018年采访中表述）：
> "基本上我这一生投了3家公司。一个是伯克希尔，一个是好市多，最后一个就是与李录合作的基金。我已经95岁了。就这三个。你不需要投资很多东西就会变得富有。"
> — [[sources/li/【文章目录】/1-李录核心文字稿/03-2018-李录-红周刊采访|03-2018-李录-红周刊采访]]（2018年，芒格）

### 格雷厄姆→巴菲特/芒格→李录的传承链

李录明确阐述了这条传承线：格雷厄姆创立了价值投资的基本框架（烟蒂股、安全边际、市场先生）；芒格推动巴菲特从格雷厄姆体系进化到关注优秀企业的护城河（常劲的表述："从猿变成人"）；李录在此基础上加入了文明三段论的宏观框架和能力圈的第四原则。

### 巴菲特入股比亚迪（2008年）

2008年，李录推动巴菲特/伯克希尔以8港元入股比亚迪。保密严格，仅3-4人知情。李录将此定性为超越资本意义的事件：

> "这就是一个共赢的事情，对于伯克希尔是一个共赢的事情，对比亚迪是一个共赢的事，对于中国的资本市场是一个共赢的事情，让世界第一次了解到，在中国正在发生神奇性的、革命性的、技术性的变化。"
> — [[sources/li/【文章目录】/1-李录核心文字稿/10-李录-首次公开对话比亚迪|10-李录-首次公开对话比亚迪]]（约2025年，李录）

### 芒格去世的最后时光（李录，2026年）

李录在2026年北大演讲中描述了芒格在感恩节周末的最后时光：周四晚上与家人吃火鸡，周日平静离世，直至生命最后一刻仍在研究股票（99岁时买入人生最后一只股票并亲眼看到翻倍）。

### 常劲与芒格思想的传播

常劲在芒格书院的参与、对芒格人格的亲身描述，以及在北大课程中系统传授芒格框架（跨学科多元思维模型、25种认知偏见、Lollapalooza效应），是喜马拉雅思想传承的重要组成部分。

常劲对芒格人格的描述（2021年芒格书院读书会）：
> "他的诚实是一种极致的诚实，不会为了别人装模作样，也不会太在乎别人对他的感受怎样。可最重要的是，他确实也会去在乎他周边的这些人，对他们非常好，好到你想象不到的地步，非常为别人着想，非常热心，非常慷慨。"
> — [[sources/li/【文章目录】/2-常劲系列/11-2021-常劲-要避免短期思维方式（转载）|11-2021-常劲-要避免短期思维方式（转载）]]（2021年，常劲）

## 时间演变

| 年份 | 事件 |
|---|---|
| 约1990年代初 | 李录误入巴菲特哥伦比亚演讲，进入价值投资 |
| 2003年 | 李录与芒格建立合伙关系，芒格家族开始通过喜马拉雅持有比亚迪 |
| 2006年 | 李录受Bruce Greenwald（巴菲特传道者）邀请，在哥伦比亚商学院演讲 |
| 2008年 | 李录推动伯克希尔8港元入股比亚迪，巴菲特首次公开背书中国企业 |
| 2018年 | 李录与芒格在奥马哈接受《红周刊》联合采访，芒格表示对李录的投资是人生三笔中的第三笔 |
| 2023年 | 芒格去世（2023年11月，感恩节周末） |
| 2025年 | 常劲在奥马哈峰会描述巴菲特传承；描述伯克希尔持有3000亿现金的逻辑 |
| 2026年 | 李录在北大演讲中描述芒格最后时光，称其影响"不会因去世而消失" |

## 内部矛盾或复杂性

李录和常劲均强调，模仿巴菲特/芒格的方法容易，模仿其对个人利益的淡泊极难——"很多人羡慕他们的成功，但绝对不愿意模仿他们对工资、对财富的态度"。这揭示了价值投资"知易行难"的核心悖论。
